PostTracker - Mobilny System Zarządzania Korespondencją

Opis Projektu

Aplikacja mobilna (React Native) służąca do zarządzania wewnętrznym obiegiem poczty i przesyłek w organizacji. Celem projektu jest zastąpienie tradycyjnych, papierowych dzienników korespondencji nowoczesnym, cyfrowym rozwiązaniem.

Aplikacja pozwoli na szybkie rejestrowanie przesyłek (za pomocą skanera kodów), śledzenie ich statusu (np. "Zarejestrowana", "Odebrana") oraz łatwą archiwizację.

Projekt realizowany jest w ramach zaliczenia przedmiotu Aplikacje Mobilne Projekt.

Główne Technologie

React Native (CLI lub Expo)

React Navigation (dla nawigacji Stack i Tab)

[React Native Paper / NativeBase] (Framework UI)

API: Własny backend nest.js do obsługi logiki biznesowej.
